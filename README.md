# Login Module - Java Servlet & JSP

A simple login system built with Java Servlets and JSP.

## Features
- User login with session management
- Protected pages with session check
- Logout functionality

## Tech Stack
- Java Servlets
- JSP
- Apache Tomcat

## How to Run
1. Clone the repo
2. Import into Eclipse as a Dynamic Web Project
3. Run on Apache Tomcat server
